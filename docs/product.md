# Product

## 目的

複数の GitHub リポジトリにまたがる Issue と Pull Request を一覧表示し、進捗管理を効率化する個人向けビューアアプリ。

## 対象ユーザー

- 複数の GitHub リポジトリを管理・参加している個人開発者
- プライベートリポジトリを含む Issue / PR を横断的に確認したいユーザー

> 機能の一覧は [README.md](../README.md) を参照。

## 成功指標

- 設定したリポジトリの Issue / PR が正しく一覧表示される
- プライベートリポジトリも PAT 権限があれば表示される
- フィルタリングが正常に動作する
- キャッシュにより2回目以降の表示が高速になる
- GitHub Pages（サブパス付き）および `npm run dev` の両方で動作する
